## Keyboard Shortcuts
This is based on my i3wm shortcuts

## Basic Shortcuts
- New Panel `alt+enter`
- Command Pallete `alt+space`
- Switch Panel `alt+[arrow keys]`
- Resize Panel `alt+shift+[arrow keys]`
- Move Panel `alt+shift+[i, j, k, l]`
- Focus Mode `alt+f`
- New Tab `alt+t`
- Next Tab `ctrl+tab`
- Select Tab `alt+[1-9]`
- Move Current Panel to Tab `alt+shift+[1-9]`
